⚡ How to Use These Templates

1. Open in Browser

Double-click any .html file in the sites/ folder. No server needed.

2. Customize

Edit the business name, address, phone, hours, and --primary-color CSS var at the top of the file.

3. Share Demo Link

Drop the file on Netlify Drop for a shareable URL to send to the client.
